Ng-Select

Is a 3rd-party all in one UI Select or Multiselect component called ng-select

Install

To install ng-select via npm run the following.

Import

Import the scss file to your project.

Usage

Add the NgSelectModule and angular FormsModule to AppModule or SharedModule.
