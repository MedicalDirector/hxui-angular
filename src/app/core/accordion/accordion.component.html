API Reference

Accordion Component
Inputs Type Description
expanded boolean Determines if the accordion container is expanded by default on load.
cssClass string Determines if the accordion container contains an additional class, binding via [ngClass].
position number Provides an index to the accordion component for each container.

Outputs Type Description
headerClick number Used in conjunction with [position]; This will emit an event with the [position] number provided.
