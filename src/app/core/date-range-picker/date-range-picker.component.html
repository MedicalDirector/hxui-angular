API Reference

DateRangePickerComponent
Inputs Type Description
dateFormat string A JavaScript Date object formatting string, formats the display of components current value. Defaults to "dd/MM/y".
placeholder string This attribute specifies the placeholder value of the components input element. Defaults to "Date".
disabled boolean Adds the disabled html attribute to the components dropdown element. Defaults to false.
displayMode number This attribute specified how tab(s) will be displayed. Options are: 1 - Show both tabs(default); 2 - Show Custom Selection only; 3 - Show Interval Selection only. Defaults to 1.
autoClose boolean Indicates that dropdown will be closed on item or document click, and after pressing ESC. Defaults to true.
Placement string Placement of a popover. Accepts: "top", "bottom", "left", "right".
intervalOptions string[] This attribute specifies interval options displayed under interval selection tab. Please see available options as below.
defaultDateRange DateRange Default date range initially displayed in date range picker. Default to today's date.

Last Fortnight Last Fortnight
Next Fortnight Next Fortnight
Outputs Description
onDateRangeChange Emits a Date Range Object containing fromDate and toDate selected from the DateRangePicker. Need to import data type: DateRange.
