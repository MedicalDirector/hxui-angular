API Reference

DateRangePickerComponent
Inputs Type Description Defaults
intervalOptions string[] Specifies interval options displayed under interval selection tab. Please see available options as below.
placeholder string Specifies the placeholder value of the components input element. 'Date'
disabled boolean Adds the disabled html attribute to the components dropdown element. false
autoClose boolean Indicates that dropdown will be closed on item or document click, and after pressing ESC. true
placement 'top' | 'bottom' | 'left' | 'right' Specifies the position the datepicker opens against the input element 'bottom'
displayMode DisplayModeEnum Specifies how tab(s) will be displayed. DisplayModeEnum.showTab
dateFormat string A JavaScript Date object formatting string, formats the display of components current value. dd/MM/yyyy
defaultDateRange DateRangeInterface Default date range initially displayed in date range picker. (today)
showCaretDown boolean Specifies whether caret down icon is displayed to right of input true
Outputs Description
onDateRangeChange Emits a Date Range Object containing fromDate and toDate selected from the DateRangePicker. Need to import data type: DateRangeInterface.
DisplayModeEnum
Enum Description
showTab Show both tabs
showCustomOnly Show Custom Selection only
showIntervalOnly Show Interval Selection only
DateRangeInterface
Interface Type

toDate Date
