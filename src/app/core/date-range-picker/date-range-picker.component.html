API Reference

DateRangePickerComponent Component
Inputs Type Description
dateFormat string A JavaScript Date object formatting string, formats the display of components current value. Defaults to "dd/MM/y".
placeholder string This attribute specifies the placeholder value of the components input element. Defaults to "Date".
disabled boolean Adds the disabled html attribute to the components dropdown element. Defaults to false.
required boolean Adds the required html attribute to the components input element and a required asterisk on the input label. Defaults to true.
align "top" | "bottom" Specifies to open the datepicker at the top of the input element or at the bottom
displayMode number To choose from: 1 - Show tabs(default) 2 - Show Custom Selection only 3 - Show Interval Selection only
autoClose boolean Indicates that dropdown will be closed on item or document click, and after pressing ESC
Placement string Placement of a popover. Accepts: "top", "bottom", "left", "right"
intervalOptions string[] This attribute enalble customisation of interval options displayed in interval tab. Options refer to a pre-defined list as below
INTERVAL OPTIONS DESCRIPTION
Today Today
Yesterdy Yesterdy
Tomorrow Tomorrow
Last Year Last Year
Next Year Next Year
Last 6 Months Last 6 Months
Next 6 Months Next 6 Months
Last 3 Months Last 3 Months
Next 3 Months Next 3 Months
Last Month Last Month
Next Month Next Month
Last Week Last Week
Next Week Next Week
Last Fortnight Last Fortnight
Next Fortnight Next Fortnight
Outputs Description
onDateChange Emits a Date is selected from the Datepicker or a valid date string is entered into input field
