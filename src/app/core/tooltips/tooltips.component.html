Tooltips

The tooltip component is a message component which appears when a cursor is positioned over an icon, image, hyperlink, or other element.

Usage

Add the TooltipsModule to AppModule or SharedModule.

Example

API Reference

TooltipContentComponent
Input Type Description
hostElement HTMLElement the element that will call this component to show
content String the content that shows inside the tooltip
placement String place the tooltip either top|bottom|left|right
context Enum set the context (colour) to either Success|Warning|Danger|None
TooltipDirective
Input Type Description
content String the content that shows inside the tooltip
disabled Boolean disable the tooltip or not
placement String place the tooltip either top|bottom|left|right
