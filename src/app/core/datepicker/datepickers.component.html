Date Pickers

The date picker is a form element that is used to select a date

Usage

Add the DatepickerModule to AppModule or SharedModule.
The date picker makes use of an excellent 3rd party angular directive called ngx-mask.
You will need to npm install ngx-mask into your application, in order to use the date picker component.

API Reference

Datepicker-Form Component
Inputs Type Description
dateFormat string A JavaScript Date object formatting string, formats the display of components current value. Defaults to "dd/MM/y".
defaultToPresentDate boolean Initializes the component with a value of the present date. Defaults to true.
placeholder string This attribute specifies the placeholder value of the components input element. Defaults to "Date".
helpText string This attribute specifies the text value of input helper. Defaults to "Please select a date".
icon string This attribute specifies the font icon name. Defaults to "hx-icon icon-calendar".
iconPosition string This attribute specifies the icon position. Defaults to "right".
disabled boolean Adds the disabled html attribute to the components input element. Defaults to false.
readonly boolean Adds the readonly html attribute to the components input element. Defaults to false.
required boolean Adds the required html attribute to the components input element and a required asterisk on the input label. Defaults to true.
allowPreviousDates boolean Setting to false will disallow the user from selecting dates before the present date. Defaults to true.
allowFutureDates boolean Setting to false will disallow the user from selecting dates after the present date. Defaults to true.
align "top" | "bottom" Specifies to open the datepicker at the top of the input element or at the bottom
from "dd/mm/yyyy" Specifies the inclusive beginning date for allowed date values
to "dd/mm/yyyy" Specifies the inclusive end date for allowed date values
interval boolean Enables interval selection. Defaults to "false"
Outputs Description
onDateChange Emits a Date is selected from the Datepicker or a valid date string is entered into input field
onFocus Emits a boolean if date picker input field is in focus
