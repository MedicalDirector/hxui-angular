Install Guide

Adding HxUI-angular into a project

1. Use NPM:



    npm install git+https://bitbucket.org/md-design/angular-hxui.git --save

You can run this command if you are using SSH method to access repo.



      npm install git+ssh://bitbucket.org/md-design/angular-hxui.git --save

2. Import all web component modules into app module or shared module.


    import {{ '{' }}HxUiModule} from "@hxui/angular/src/modules/index";

    @NgModule({{ '{' }}
      declarations: [
        AppComponent
      ],
      imports: [
        HxUiModule.forRoot()
      ],
      providers: [],
      bootstrap: [AppComponent]
    })
    export class AppModule {{ '{' }}

3. Alternatively import each web component module that you need individually.


      import {{ '{' }}DropdownModule} from "@hxui/angular/src/modules/dropdown/dropdown.module";

      @NgModule({{ '{' }}
        imports: [DropdownModule.forRoot(),...]
      })
      export class AppModule(){{ '{' }}
