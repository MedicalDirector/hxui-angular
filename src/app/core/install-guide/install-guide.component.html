Install Guide

Adding HxUI-angular into a project

1. Use NPM:

You can run this command if you are using personal access tokens(https) method to access Visual Studio Team Services.



    npm install git+https://medicaldirector.visualstudio.com/MD%20Design/_git/HxUi-angular --save

For Mac users, you need to install Git Credential Manager for Mac in order to access Visual Studio Team Services properly.


You can run this command if you are using SSH method to access Visual Studio Team Services.



    npm install git+ssh://medicaldirector@medicaldirector.visualstudio.com:22/MD%20Design/_git/HxUi-angular --save

You may need to add SSH public keys to Visual Studio Team Services in order to install HxUI using npm.


2. Import all web component modules into app module or shared module.


    import {{ '{' }}HxUiModule} from "@hxui/angular/src/modules/index";

    @NgModule({{ '{' }}
      declarations: [
        AppComponent
      ],
      imports: [
        HxUiModule.forRoot()
      ],
      providers: [],
      bootstrap: [AppComponent]
    })
    export class AppModule {{ '{' }}

3. Alternatively import each web component module that you need individually.


      import {{ '{' }}DropdownModule} from "@hxui/angular/src/modules/dropdown/dropdown.module";

      @NgModule({{ '{' }}
        imports: [DropdownModule.forRoot(),...]
      })
      export class AppModule(){{ '{' }}
