Inspector

The inspector mainly used for displaying, forms, details or previews in a full height panel that comes in from the left or right.

Usage

Add the InspectorModule to AppModule or SharedModule.

Example


Code

You need to declare the custom modal component as an entry component for dynamic loading

API Reference

InspectorService
Method Type Description
open(component: any, config: InspectorConfig, parameters?: Object) InspectorOverlayRef Opens and loads a dynamic component
InspectorConfig
Property Type Description Default
panelClass string | string[] OPTIONAL Opens and loads a dynamic component
hasBackdrop boolean OPTIONAL Does the inspector have a backdrop effect? true
backdropClass string OPTIONAL Custom styling class for backdrop 'hx-modal-background'
closeOnBackdropClick boolean OPTIONAL Automatically close inspector on backdrop click true
size InspectorSize OPTIONAL Determines width of inspector InspectorSize.Small
location InspectorLocation OPTIONAL Determines side of screen inspector is anchored to InspectorLocation.Right
hasClose boolean OPTIONAL Does inspector have exterior close icon? true
