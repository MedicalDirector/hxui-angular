API Reference

TabsetComponent
Inputs Type Description Default
justified boolean if true tabs fill the container and have a consistent width
type string navigation context class: 'tabs' or 'pills'
vertical boolean if true tabs will be placed vertically
hasInfo boolean if true info tab classes will be added
tag 'link'|'button' apply the html element that makes sense semantically 'link'
contentCustomClass string if set, will be added to the tab's content class attribute
changeFn Promise<boolean> Provides a hook to intercept the changing of Promise<true>
stickyHeader boolean if set, will make the tab nav items sticky
stickyHeaderOffset string Add an offset (usually negative) to the position of sticky. REM is the unit of value.
TabDirective
Inputs Type Description
active boolean tab active state toggle
customClass string if set, will be added to the tab's class atribute
disabled boolean if true tab can not be activated
heading string tab header text
id string tab id
removable boolean if true tab can be removable, additional button will appear
Outputs Description
deselect fired when tab became inactive, $event:Tab equals to deselected instance of Tab component
removed fired before tab will be removed, $event:Tab equals to instance of removed tab
select fired when tab became active, $event:Tab equals to selected instance of Tab component
TabsetConfig
Properties Type Description
type string provides default navigation context class: 'tabs' or 'pills'
