options IFilterOption[] OPTIONAL Filter options (for type FilterType.SingleSelect)
value string OPTIONAL Filter value (for types FilterType.Search and FilterType.DateRange)
sourceValue any OPTIONAL Original value derived from source component. (e.g string for search type, DateRange for DateRangePicker type)
