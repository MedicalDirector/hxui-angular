API Reference

Filters Component
Inputs Type Description
collapsed boolean Toggles the collapsed state of the filter component. Defaults to false.
filters IFilterConfig[] An array of filter configurations
IFilterConfig
Property Type Description
id number Unique identifier
type FilterType Filter type, can be SingleSelect, MultiSelect or Search
label string Label used for display purposes.
options IFilterOption[] OPTIONAL Filter options for Single Select and MultiSelect types
value string OPTIONAL Filter value for Search filter type
defaultIndex number Index of default filter option. The option to set when filters are cleared.
callback Function Function to call when filter selected.
IFilterOption
Property Type Description
label string The label used for display purposes
value any The value used when filter option selected
selected boolean Used to determine if one or more options selected. Defaults to false.
