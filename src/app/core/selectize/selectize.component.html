API Reference

SelectizeComponent
Input Type Description
config SelectizeConfig Configuration for Selectize instance. Read the Selectize documentation for a full list of config options.
options ISelectizeItem[] An array of the initial options available to select; array of objects. Read the Selectize documentation for a full list of config options.
optgroups any[] Option groups that options will be bucketed into. Make sure each object in the array has a property named whatever optgroupValueField is set to.
id string Input element id
placeholder string The placeholder of the control (displayed when nothing is selected / typed).
enabled boolean Disable or enable input element
value ISelectizeItem[] Selected value(s)
formControl FormControl Reference to a FormControl
errorClass string CSS class to be appended on error

Output Type Description
onFocus EventEmitter Event fired when input has focus.
onBlur EventEmitter Event fired when input losses focus.
onOptionAdded EventEmitter Event fired when a new option is added. Returns the added option.
SelectizeConfig
Property Type Default Description
label string 
Selectize Label
 The form label text
help string 
Selectize help text
 Helper text shown on focus or error
hasCaret boolean 
false
 Force caret to show. Only applies to multi select.
mandatory boolean 
false
 Mandatory boolean controls whether the is shown in the label or not. Has nothing to do with formControl.
delimiter string 
','
 The string to separate items by. When typing an item in a multi-selection control allowing creation, then the delimiter, the item is added. If you paste delimiter-separated items in such control, the items are added at once. The delimiter is also used in the getValue API call on a text input tag to separate the multiple values.
create boolean|function false Allows the user to create new items that aren't in the initial list of options. This setting can be any of the following: true, false (disabled), or a function to process input. The function can take one of two forms: synchronous (with signature function(input){{'{'}}} or asynchronous (with signature function(input, callback). In the synchronous case, the function should return an object for the options (eg, with defaults: return {{'{'}} 'value': value, 'text': text };). The asynchronous version should invoke the callback with the result in the same format as the object above (eg, callback( {{'{'}} 'value': value, 'text': text});)
createOnBlur boolean false If true, when user exits the field (clicks outside of input), a new option is created and selected (if create setting is enabled).
createFilter RegExp|string|function null Specifies a RegExp or a string containing a regular expression that the current search filter must match to be allowed to be created. May also be a predicate function that takes the filter text and returns whether it is allowed.
highlight boolean true Toggles match highlighting within the dropdown menu.
persist boolean true If false, items created by the user will not show up as available options once they are unselected.
maxOptions int 1000 The max number of items to render at once in the dropdown list of options.
maxItems int 1 The max number of items the user can select. 1 makes the control mono-selection, null allows an unlimited number of items.
hideSelected boolean false If true, the items that are currently selected will not be shown in the dropdown list of available options.
closeAfterSelect boolean false If true, the dropdown will be closed after a selection is made.
allowEmptyOption boolean false If true, Selectize will treat any options with a "" value like normal. This defaults to false to accomodate the common select practice of having the first empty option to act as a placeholder.
placeholder string undefined The placeholder of the control (displayed when nothing is selected / typed). Defaults to input element's placeholder, unless this one is specified.
dropdownParent string null The element the dropdown menu is appended to. This should be 'body' or null. If null, the dropdown will be appended as a child of the Selectize control.
addPrecedence boolean false If true, the "Add..." option is the default selection in the dropdown.
