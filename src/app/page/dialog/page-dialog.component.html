Dialogs

A dialog mainly used for adding form content or showing a notification of some sort.

Usage

Add the DialogModule to AppModule or FeatureModule.

If lazy-loaded, add the following to the LazyFeatureModule.

Basic example

You need to declare the custom modal component as an entry component for dynamic loading
