Time Picker

The time picker is a form element that is used to select a time

This component is only available for version 11+.

Usage

If not importing all HXUI modules globally, add TimepickerModule to imports in AppModule or MyFeatureModule.

Examples


meridian: true
size: medium
meridian: true
size: small
meridian: true
size: large
meridian: false
size: small
meridian: false
size: medium
spinners: false
meridian: false
size: medium
seconds: true
Form value object: {{ form.get("time").value | json }}

API reference


TimepickerComponent


Inputs Type Description Defaults
meridian boolean Whether to display 12H (true) or 24H (false) mode false
spinners boolean If `true`, the spinners above and below inputs are visible. true
seconds boolean If `true`, it is possible to select seconds. false
hourStep number The number of hours to add/subtract when clicking hour spinners. 1
minuteStep number The number of minutes to add/subtract when clicking minute spinners. 1
secondStep number The number of seconds to add/subtract when clicking second spinners. 1
disabled boolean Whether the timepicker input is disabled false
readonlyInputs boolean If `true`, the time-picker is readonly and can't be changed. false
size 'small' | 'medium' | 'large' The size of inputs and buttons. 'medium'
