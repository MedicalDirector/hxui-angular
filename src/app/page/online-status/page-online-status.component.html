Online Status

The online status service detects when an angular app goes offline/online

Usage

Add the HxUiModule to AppModule or SharedModule.

Basic example

Use the browser console to emulate offline mode.

App is currently:

{{ (isOnline)? 'Online' : 'Offline' }}

API reference

OnlineStatusService

Properties Type Description
online Observable Fires when online mode detected
offline Observable Fires when offline mode detected
