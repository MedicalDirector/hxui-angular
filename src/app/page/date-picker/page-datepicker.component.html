Date Picker

The date picker is a form element that is used to select a date

Usage

Add the DatepickerModule to AppModule or SharedModule.

If lazy-loaded, add the following to the LazyFeatureModule.
The date picker makes use of an excellent 3rd party angular directive called ngx-mask .
You will need to npm install ngx-mask into your application, in order to use the date picker component.

API reference

DatepickerFormComponent

Inputs Type Description Defaults
id string ''
disabled boolean Adds the disabled html attribute to the components input element. false
readonly boolean Adds the readonly html attribute to the components input element. false
required boolean Adds the required html attribute to the components input element and a required asterisk on the input label. true
defaultToPresentDate boolean Initializes the component with a value of the present date. true
allowPreviousDates boolean Setting to false will disallow the user from selecting dates before the present date. true
allowFutureDates boolean Setting to false will disallow the user from selecting dates after the present date. true
dateFormat string A JavaScript Date object formatting string, formats the display of components current value. 'DD/MM/Y'
placeholder string This attribute specifies the placeholder value of the components input element. 'Date'
helpText string This attribute specifies the text value of input helper. 'Please select a valid date'
helpTextVisible string Specifies visibility of input help text false
isWarning boolean Warning state of input false
isDanger boolean Danger state of input false
icon string This attribute specifies the font icon name. 'hx-icon icon-calendar'
iconPlacement string This attribute specifies the icon placement. 'right'
placement 'top' | 'bottom' | 'left' | 'right' Specifies the position the datepicker opens against the input element 'bottom'
showDelay number delay in ms before showing the calendar after show is called 0
hideDelay number delay in ms before hiding the calendar after hide is called 0
from "dd/mm/yyyy" Specifies the inclusive beginning date for allowed date values
to "dd/mm/yyyy" Specifies the inclusive end date for allowed date values
interval boolean Enables interval selection. false
dueDateInterval string '0 day(s)'
maskPattern string Mask pattern for date picker text input 'd0/M0/0000'
Outputs Description
dateChange Emits a Date is selected from the Datepicker or a valid date string is entered into input field
inputFocus Emits a boolean if date picker input field is in focus
