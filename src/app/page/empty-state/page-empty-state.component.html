Empty State

The default state of a specific area, module, component is usually filled with a so called empty state.
It's a component that requires users to go through one or more steps to populate it with data.

Usage

Add the EmptyStateModule to AppModule or SharedModule.

Basic example

API reference

EmptyStateComponent

Inputs Type Description
icon string A string value of the icon name.
msg string The message to describe why the state is empty.
