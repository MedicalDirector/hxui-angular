API reference

Accordion Component

Inputs Type Description
expanded boolean Determines if the accordion container is expanded by default on load.
cssClass string Determines if the accordion container contains an additional class, binding via [ngClass].
index number Provides an index to the accordion component for each container.
disabled boolean False by default. Disables the accordion from expanding on any row which fits the condition. (headerClick) will still emit an output if configured.
empty boolean True by default. Hides the caret down/up if the accordion container is empty.

Outputs Type Description
headerClick number Used in conjunction with [index]; This will emit an event with the [index] number provided.
